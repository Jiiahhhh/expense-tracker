# 💸 Expense Tracker

A simple expense tracking app built with Vanilla JavaScript and ES6 Modules.

## ✨ Features

- Add and delete expenses with categories
- Real-time summary by category
- Filter expenses by category
- Data persisted in localStorage
- Export expenses to JSON file
- Import expenses from JSON file

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- ES6 Modules (import/export)
- localStorage for data persistence
- Error handling with try/catch
